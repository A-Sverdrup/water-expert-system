### Requirements:

Free-threaded versions areincompatible with required dependency ```pandas```.


### Dependencies:

```
python3 -m pip install numpy
python3 -m pip install pandas=2.2.3
python3 -m pip install Bio
```

7-zip: 7z.exe / 7zz (included, optional)
